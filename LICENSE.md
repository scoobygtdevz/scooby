## 🔴 Troubleshooting & Common Issues

### Menu Doesn't Open In-Game
**Cause:** Injection failed silently or keybind conflict
**Fix:**
```bash
# Run loader with verbose logging to identify the issue
./ScoobyLoader.exe --verbose --log-level=debug
```
Then check `~/.scooby/logs/latest.log` for injection error codes.

### Antivirus Blocking the File
**Cause:** Heuristic detection false positive — memory injectors trip AV scanners
**Fix:** Add the Scooby folder to your AV exclusion list before extracting the archive.

### iOS App Crashes on Launch
**Cause:** Certificate expired (AltStore-signed apps expire every 7 days)
**Fix:** Re-sign the IPA via AltStore — open AltStore → My Apps → long-press Scooby → Refresh

### Config Not Saving
**Cause:** File permissions issue on the config directory
```bash
# Fix permissions on macOS/Linux
chmod 755 ~/.scooby/config/
chmod 644 ~/.scooby/config/scooby_config.toml
```

### Android Overlay Not Appearing
**Cause:** Overlay permission not granted properly
**Fix:** Go to Settings → Apps → Scooby → Display over other apps → Enable
